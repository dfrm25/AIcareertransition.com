You're Probably Using the Wrong AI Model. Here's Where to Change It.

Default = often the cheapest, not the best. Same prompt, different model = different quality. Where to change it in Copilot, ChatGPT, Claude, and Gemini—below.

If cost isn't an issue, use the latest model.

Newer model = better results. The UI moves around between updates, but the pattern is the same: find the model name near the top of chat and click it. Model names as of March 2026 are in the table below.

Microsoft Copilot

Web (copilot.microsoft.com) or inside Word/Outlook via the sidebar.

  1. Open Copilot at copilot.microsoft.com or inside a Microsoft 365 app.
  2. Look for a dropdown near the chat input — it shows the current mode (Balanced, Creative, Precise) or a model name.
  3. Select Creative for writing tasks or the highest-tier model if your plan includes it.

ChatGPT

The model name sits at the top of every chat. Free = limited options; Plus/Pro = full picker.

  1. Open chatgpt.com and start or open a chat.
  2. Click the model name at the top-centre of the chat window — it's usually "ChatGPT" with a dropdown arrow.
  3. Choose the Thinking option for complex tasks; the standard mode for speed.
chatgpt.com

[ Chat window — top-centre ]
─────────────────────────────────────
ChatGPT ▼ ← click here
• ChatGPT (standard — fast)
• ChatGPT Thinking ← use for complex tasks
( picker appears at the top of the new-chat screen or in the model header )

Claude

claude.ai. Model selector near the top of chat or in the left sidebar.

  1. Open claude.ai.
  2. Find the model name near the top of chat or in the sidebar — it shows the current Claude version.
  3. Choose Claude Sonnet for everyday tasks, Claude Opus for complex or long documents. Enable Extended Thinking in the same menu for reasoning-heavy tasks.
claude.ai

[ New chat — top of screen ]
─────────────────────────────────────
Claude Sonnet ▼ ← click here
• Claude Sonnet (fast, general use)
• Claude Opus ← complex reasoning
• Extended thinking toggle (same panel)

Gemini

gemini.google.com. Model name near the top or in a "Try Gemini Advanced" prompt.

  1. Open gemini.google.com.
  2. Look for the model name or "Gemini Advanced" option near the top of the page.
  3. Use Gemini 2.0 Flash for everyday tasks; enable Deep Research mode (in the + menu near the input) for research-heavy queries.

Quick comparison

Tool Best quality option (March 2026) Where to find it
Copilot Creative mode or latest model in dropdown Dropdown near chat input
ChatGPT ChatGPT Thinking (reasoning) / standard (fast) Model name at top of chat
Claude Claude Opus + Extended Thinking toggle Model picker top of chat / sidebar
Gemini Gemini Advanced / 2.0 Flash Model dropdown at top of page

UI layouts change with every update, but the pattern stays the same: model name near the top of chat → click → pick the best one available on your plan. Check it before you start a complex task.
