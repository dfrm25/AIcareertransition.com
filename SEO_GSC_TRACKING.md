## Weekly Metrics

Record these from Google Search Console:

- Query
- Page
- Impressions
- Clicks
- CTR
- Average position
- Indexed status
- Canonical status
- Notes / action

## Decision Rules

- If impressions rise but CTR is under 2%, rewrite title/meta for the ranking page.
- If position is 11-20, add internal links from the homepage, blog hub, and related posts.
- If position is 21-50, expand the page section matching the query shown in GSC.
- If the page is crawled but not indexed, improve unique examples, add internal links, and request indexing.
- If queries are unrelated, tighten headings and introductory copy around the intended search intent.

## Monthly Actions

- Refresh `sitemap.xml` after publishing or updating pages.
- Add 4-8 internal links into pages with impressions.
- Run 40 qualified outreach touches using `BACKLINK_OUTREACH_PLAYBOOK.md`.
- Update the pillar page with new questions from GSC query data.
